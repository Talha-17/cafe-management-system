# 🍽️ Python Restaurant System

This is a simple Python project that simulates a small restaurant ordering system.  
You can choose items from the menu, and it will calculate your total bill.

---

## 💡 What this project does

Basically, it works like a mini restaurant cashier system:

- Shows you a menu with prices
- Lets you order food items
- Checks if the item exists in the menu
- Adds up your total bill
- Shows the final amount at the end

---

## 🍕 Menu

- Pizza — Rs 40  
- Pasta — Rs 50  
- Burger — Rs 60  
- Salad — Rs 70  
- Coffee — Rs 80

## ▶️ How to run it

Just follow these steps:

1. Make sure Python is installed on your system
2. Save the file as `restaurant.py`
3. Open terminal or command prompt in that folder
4. Run this command:

```bash
python restaurant.py
